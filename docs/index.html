Bienvenido a GiConnect

La aplicación más completa para la gestion de los centros de artes marciales.

Frontend (GiConnect)

  1. Framework Principal:
    • Ionic Framework (v8.0.0)
    • Angular (v19.0.0)
  2. Lenguajes de Programación:
    • TypeScript
    • HTML/CSS
  3. Componentes y Bibliotecas:
    • Angular Material (v19.2.14) para componentes UI
    • Capacitor (v7.2.0) para funcionalidades nativas
    • Firebase (v11.9.1) para servicios en la nube
    • Socket.IO Client para comunicación en tiempo real
    • Moment.js para manejo de fechas
    • RxJS para programación reactiva
  4. Herramientas de Desarrollo:
    • Angular CLI para gestión de proyectos
    • ESLint para linting
    • Karma y Jasmine para testing

Backend (GiBack)

  1. Framework y Runtime:
    • Node.js (>=14.0.0)
    • Express.js (v5.1.0)
  2. Base de Datos:
    • MongoDB (usando Mongoose v8.15.0 como ODM)
  3. Autenticación y Seguridad:
    • JWT (jsonwebtoken v9.0.2)
    • Bcrypt para encriptación de contraseñas
  4. Características Adicionales:
    • Nodemailer para envío de emails
    • Express Validator para validación de datos
    • CORS para manejo de peticiones cross-origin
    • Swagger para documentación de API
    • Node-cron para tareas programadas
  5. Despliegue:
    • Railway (evidenciado por railway.json)
    • Procfile para configuración de despliegue

Características Técnicas Destacables

  1. Arquitectura:
    • Arquitectura cliente-servidor
    • API RESTful
    • Comunicación en tiempo real con Socket.IO
  2. Seguridad:
    • Autenticación basada en JWT
    • Encriptación de contraseñas con Bcrypt
    • Validación de datos con Express Validator
    • Manejo seguro de sesiones
  3. Desarrollo:
    • Documentación con JSDoc
    • Testing automatizado con Karma y Jasmine
    • Linting y formateo de código con ESLint
    • Control de versiones con Git

Esta stack tecnológica es moderna y robusta, permitiendo:

  1. Desarrollo multiplataforma (web, Android, iOS)
  2. Escalabilidad
  3. Mantenibilidad
  4. Seguridad
  5. Experiencia de usuario fluida
  6. Comunicación en tiempo real
  7. Documentación completa
